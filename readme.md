# Joan Gordo - Currículum Vitae Web

## Descripció
Aquest projecte consisteix en un **Currículum Vitae interactiu i responsive** de Joan Gordo, desenvolupat amb **HTML, CSS i Font Awesome**. El CV està dissenyat per a **mòbils, tablets i ordinadors**, amb un **sidebar fix** en pantalles grans i adaptacions específiques per a impressió.  

L’objectiu és presentar de manera clara i professional la **formació, experiència, habilitats i projectes**, amb una **interfície visual agradable i accessible**.

## Tecnologies utilitzades
- **HTML5** – Estructura semàntica del contingut.  
- **CSS3** – Estils visuals, adaptació responsive i efectes hover.  
- **Font Awesome 6** – Icones per a seccions, habilitats i contacte.  
- **Responsive Design** – Mobile-first amb `@media queries` per a tablet i ordinador.  

Opcional:  
- Favicon i fotografia de perfil inclosos.  
- Adaptació per a impressió (`@media print`) per generar un PDF net del CV.

## Seccions del CV
1. **Sobre Mi** – Breu presentació i experiència inicial.  
2. **Experiència** – Experiències laborals i pràctiques rellevants.  
3. **Formació** – Estudis de grau mitjà i superior.  
4. **Habilitats** – Competències tècniques amb sistema de valoració amb estrelles.  
5. **Projectes** – Projectes acadèmics i pràctics amb enllaços externs.  
6. **Idiomes** – Nivell de domini amb barres de progrés.  
7. **Contacte** – Email, telèfon, GitHub i LinkedIn.  

> En pantalles grans, les seccions **Idiomes** i **Contacte** es mostren en un **sidebar fix**; en dispositius mòbils apareixen dins de les targetes `card-movil`.
